# pi-power-button

Scripts used in the [Pi power button guide](https://howchoo.com/g/mwnlytk3zmm/how-to-add-a-power-button-to-your-raspberry-pi) on howchoo.

## Installation

1. Clone down this repo to your Raspberry Pi
2. Run `script/install`

That's it!
